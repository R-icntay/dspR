1 Discrete-time signals

Signals are broadly classified into two categories; analog signals \(x_a (t)\) and discrete signals \(x[n]\). In discrete signals, the variable \(n\) is an integer which represents discrete instances in time, hence the alternative name discrete-time signals.

1.1 Types of sequences

There are several elementary sequences for DSP. Let’s take a brief look at their definitions and R representations:

1. Unit sample sequence

The unit sample sequence is defined as the sequence:

\[ \delta [n] = \begin{cases} 1, \ \ \ \ \ n = 0, \\ 0, \ \ \ \ \ n \neq 0. \end{cases} \]

The unit sample sequence is typically referred to as an impulse. For example, the signal

\[ \delta [n-n_0] = \begin{cases} 1, \ \ \ \ \ n = n_0, \\ 0, \ \ \ \ \ n \neq n_0. \end{cases} \tag{1}\]

over the interval \(n_1\leq n_0\leq n_2\) , can be implemented and represented in R as:

library(tidyverse)

# Function that evaluates impulse sequence
gen_impulse <- function(n0, n1, n2){
  # Generates x(n) = delta(n-n0); n1 <= n <= n2
  
  # Generate sequence 
  n = n1:n2
  # If n = n0, n-n0 = 1
  x = as.numeric((n - n0) == 0)
  
  return(tibble(n, x))
  
}

# Call function
imp_seq <- gen_impulse(n1 = -5, n0 = 0, n2 = 5)

2. Unit step sequence

The unit step sequence is defined as:

\[ u[n] = \begin{cases} 1, \ \ n\ge 0, \\ 0, \ \ n\le 0 . \end{cases} \]

Again, let’s implement and visualize the following unit step sequence:

\[ u [n - n_0] = \begin{cases} 1, \ \ n \geq n_0, \\ 0, \ \ n < n_0. \end{cases} \]

over the interval \(n_1\leq n_0\leq n_2\)

# Function that generates x(n) = u(n-n0); n1 <= n <= n2
gen_stepseq <- function(n0, n1, n2){
  n = n1:n2
  x = if_else(n >= n0, true = 1, false = 0)
  ## Alternative evaluation
  #x = if_else(n-n0 >= 0, true = 1, false = 0)
  
  return(tibble(n, x))
}

3. Real-valued exponential sequence

Exponential sequences are another important class of basic signals. Real valued exponential sequences are generally represented as:

\[ x[n] = a^n, \forall n; \ \ \ \ a \in \mathbb{R} \]

The formula simply means that if \(a\) is a real number, then the sequence is real. The sequence oscillates with an exponentially growing envelope if \(|a| > 1\) or with an exponentially decaying envelope if \(|a| < 1\)

\[ x[n] = (0.8)^n, \ \ \ -5 \leq n \leq 10 \]

  x = 0.8 ^ n

4. Complex-valued exponential sequence

Now, let’s consider the complex valued exponential sequence:

5. Sinusoidal sequence

A sinusoidal sequence can be expressed as:

\[ x(n) = Acos(\omega_0 + \theta_0), \ \ \forall n \]

where \(A\) is an amplitude, \(\theta_0\) is the phase in radians, \(\omega_0\) is the frequency in radians. For example:

\[ x(n) = 3cos(0.1\pi n + \pi/3) \ + 2sin(0.5\pi n), \ \ 0\le n \leq 10 \] can be evaluated as

# Sinusoidal sequence
sins_seq <- tibble(
  n = 0:10,
  x = 3*cos(0.1*pi*n + pi/3) + 2*sin(0.5*pi*n)
  
)

# Create stem plot of sequence
seq_plot(sins_seq) +
  ggtitle("Sinusoidal sequence")

6. Random sequences

Many practical sequences cannot be described by mathematical expressions like those above and are instead represented using random sequences. Random sequences are characterized by parameters of the associated probability density functions. There are a number of functions for generating statistical distributions available in (2021) package provided with base-R.

For instance, runif generates uniformly distributed random numbers between [0, 1] by default, while rnorm generates normally distributed random numbers with mean equal to 0 and standard deviation equal to 1 by default.

Tip

R packages tend to have very useful documentation. To find out more about a given function, say for instance randn, simply run:

help("rnorm") or

?rnorm

Other random sequences can be generated using transformations of the above functions.

7. Periodic sequences

A sequence \(x(n)\) is said to be periodic if \(x(n) = x(n + N), \forall n\). The smallest integer that satisfies this relation is called the fundamental period. Let’s illustrate this with a quick example:

Consider the signal \(x[n] = cos[\pi n/4]\)

This signal has a period

\(N = \frac{2\pi}{\omega_0} \ = \frac{2\pi}{\pi/4} \ = 8\)

Let’s evaluate if the signal is periodic by proving \(x(n + N) = x(n)\)

\(x(n + 8) = cos[\pi \frac {(n+8)}{4}] \ = cos[\pi n/4 + 2\pi]\)

using the compound formula \(cos(A+B) = cosAcosB - sinAsinB\)

\(x(n+8) = cos[\pi n/4]\)

\(\therefore the \ signal \ is \ periodic.\)

This signal has a period

Let’s evaluate if the signal is periodic by proving \(x(n + N) = x(n)\)

\(x(n + 8) = cos[\pi \frac {(n+8)}{4}] \ = cos[\pi n/4 + 2\pi]\)

using the compound formula \(cos(A+B) = cosAcosB - sinAsinB\)

\(x(n+8) = cos[\pi n/4]\)

\(\therefore the \ signal \ is \ periodic.\)
